# PulsarStream Converter

an AI generated web application (Yes, you've heard it right a vibe coded app) for converting between different data formats for class objects/models specially Java and Kotlin. This source code is generated by Claude AI (Model - Opus 4.6).

``
Note: There still some issues left to fix, for now I recommend to use only SQL to Java and SQL to Kotlin conversion vice-versa. 
``

## Features (Conversion to Class Objects/Models)

- Convert JSON to Java
- Convert JSON to Kotlin
- Convert SQL to Java
- Convert SQL to Kotlin
- Convert Java to JSON
- Convert Kotlin to JSON
- Convert ERD to Java
- Convert ERD to Kotlin
- Convert Java to ERD
- Convert Kotlin to ERD
- Convert Proto to Java
- Convert Proto to Kotlin
- Convert Java to Proto
- Convert Kotlin to Proto
- Convert GraphQL to Java
- Convert GraphQL to Kotlin
- Convert Java to GraphQL
- Convert Kotlin to GraphQL

## Usage

1. Clone the repository
```bash
git clone https://github.com/synczer0/pulsarstream.git 
```
2. Install and build dependencies
```bash
npm install
npm run build
```
3. Run the web application
```bash
npm run dev
```
4. Open http://localhost:5173 in your browser.
